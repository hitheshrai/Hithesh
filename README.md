# Hithesh

Portfolio site for AI engineering and energy materials research.

Live: https://hitheshrai.github.io/Hithesh/

## Stack

React, TypeScript, Vite, Tailwind CSS.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

```bash
npm run deploy
```
